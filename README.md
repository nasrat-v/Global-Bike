# Global-Bike
Cross platform application - Educational uses

### CityBikes public API
http://api.citybik.es/v2/

## Build
Only in dev  
  
```react-native start```

## Launch android
Start emulator or connect devide, then  
  
```react-native run-android```


## Launch iOS
Start emulator or connect devide, then  
  
```react-native run-ios```
